Story number and name: 1-Extend user profile detail with usage
Story points: 3
Brief description to see the story in action:

Story number and name: 3-Visibility control of photos
Story points: 4
Brief description to see the story in action:
The 'share with' dropdown list contains all the users along with don't specify list option on TopBar.
When any user is selected the photo and its comments are visible to both current user and selected users.
When used don't specify list then the photo and its comments is visible to everyone.
When list is empty the photo and comments are visible to only current user.

Story number and name: 5-Deleting Comments, Photos, and Users
Story points: 4
Brief description to see the story in action:
In userPhotos page there is delete button under photos and comments which deletes the respective item.
Also we have delete account at the bottom of the page that deletes user account.
Note: delete operation can only be done in their particular account.
